# Front-end

Base estática do frontend usando Bootstrap e AJAX com `fetch`.

## Estrutura

- `index.html` - layout inicial
- `assets/js/app.js` - helper de requisições AJAX
- `assets/css/style.css` - ajustes visuais mínimos

## Uso

Abra o `index.html` no navegador e ajuste `API_BASE_URL` em `assets/js/app.js` quando o backend estiver disponível.
